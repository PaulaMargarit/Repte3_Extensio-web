# Repte3_Extensió-web
Autor: **Paula Margarit Corominas**
## Descripció
Vinil d’estudi és una extensió web creada a partir de l’activitat “Rellotge creatiu” on es va crear el Vinil del Temps per representar l’hora, d’una manera abstracta i visual. Aquesta extensió adapta aquell concepte a un ús funcional: millorar la concentració i estructurar el temps d’estudi.

És una extensió web basada en Manifest V3 que funciona amb una acció de popup page, i és compatible amb Firefox i Chrome.
## Funcionament
L’extensió mesurar períodes d’una hora, que és un temps habitual de treball intens, abans de fer una pausa. El vinil gira i augmenta progressivament la velocitat a mesura que passen els minuts. També canvia de color cada 15 minuts, creant una divisió natural en quatre trams que ajuden a situar-se dins de l’hora.

L’objectiu final és oferir una eina visual, minimalista, que permeti mantenir el focus durant una hora d’estudi, convertint el temps en una experiència més tangible i agradable.

## Instal·lació
1. Descarrega el repositori.  
2. Obre el navegador (per exemple Chrome) i ves a extensions.  
3. Activa **Mode desenvolupador**.  
4. Clica **Carregar des d’una carpeta**.  
5. Selecciona la carpeta del projecte.
