# Repte3_Extensió-web
Autor: **Paula Margarit Corominas**
## Descripció
Vinil d’estudi és una extensió web creada a partir de l’activitat “Rellotge creatiu” on es va crear el Vinil del Temps per representar l’hora, d’una manera abstracta i visual. Aquesta extensió adapta aquell concepte a un ús funcional: millorar la concentració i estructurar el temps d’estudi.
